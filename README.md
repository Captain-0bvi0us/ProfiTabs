# ProfiTabs

Офлайн PWA-приложение для создания и обмена гитарными/басовыми/барабанными табулатурами.

## Возможности

- Создание табов для гитары, электрогитары, бас-гитары и барабанов
- Редактирование с клавиатуры (десктоп) и тапом (мобильные)
- Воспроизведение (playback) с визуальным курсором
- Сохранение в локальное хранилище (IndexedDB)
- Экспорт табов в `.profitabs.json` файлы
- Импорт табов из файлов
- Обмен через «Поделиться» (Web Share API) / Telegram / почту / AirDrop
- Полная работа офлайн после установки как PWA

## Быстрый старт

```bash
cd frontend
npm install
npm run dev
```

Открыть `http://localhost:5173`

## Установка на телефон (PWA)

1. Запустить `npm run build && npm run preview` (или задеплоить на любой статический хостинг)
2. Открыть сайт в Chrome на телефоне
3. Нажать «Добавить на главный экран» / «Установить приложение»
4. Готово — приложение работает офлайн

## Обмен табами

### Отправить:
1. Открыть таб → нажать **📤 Поделиться**
2. На Android: системное меню «Поделиться» (Telegram, почта, Bluetooth и т.д.)
3. На десктопе: скачивается `.profitabs.json` файл

### Получить:
1. Открыть **Мои табы** → нажать **📥 Импорт**
2. Выбрать полученный `.profitabs.json` файл
3. Таб появится в списке

## Деплой (опционально)

Приложение полностью статическое — можно разместить на любом хостинге:

```bash
cd frontend
npm run build
```

Содержимое папки `dist/` загрузить на GitHub Pages / Netlify / Vercel / любой файловый хостинг.

## Технологии

- React 18 + Vite
- IndexedDB (через `idb`) для хранения данных
- Vite PWA Plugin (Workbox) для офлайн-работы
- Web Share API для обмена файлами
